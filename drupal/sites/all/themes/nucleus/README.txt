Installation
-------------------------------------------------------------------------------
From Demo profile package:
This installation profile includes the same demo content of TB Simply at:
http://demo.themebrain.com/tb_simply/
Step 1: Extract demo profile package to your host.
Step 2: Navigate to the folder you have extracted the zip file and install
  themebrain profile.

Theme only installation:
Step 1: Extract included zip files Nucleus and TB Simply into sites/all/themes
  folder.
Step 2: Install Nucleus Theme first. Go to theme settings under "Appearence"
  section and install this theme via upload.
Step 3: Install TB Simply theme. Go to theme settings under "Appearence" section
  and Install this theme via upload.
Step 4: Enable and set as default.
